Hotness endpoint that retrieve the list of most 50 active items on the site filtered by type, returning a org.audux.bgg.response.HotList.

Requesting the top 50 'hottest' board games right now can be done as follows:

Response<HotList> hotList = BggClient.hotList(type = HotListType.BOARD_GAME).call()
println(familyItems.data) // Prints a list of HotListItem with some info about the board game

The actual data class returned is Request which can then be used to make the actual call to the API using call or callAsync. This will then return a Response and wrap a data class (on success). e.g. the above example might return a org.audux.bgg.response.HotList looking something as follows:

HotList(items = listOf(
 ListItem(
     id = 123,
     rank = 1,
     name = "The HOTTEST game",
     thumbnail = "http://...",
     yearPublished = 1999
 ),
 // +49 items...
)
