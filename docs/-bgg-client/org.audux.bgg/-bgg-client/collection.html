Request details about a user's collection.

NOTE: The default (or using subType=ThingType.BOARD_GAME) returns both ThingType.BOARD_GAME and ThingType.BOARD_GAME_EXPANSION in the collection... BUT incorrectly marks the subType as ThingType.BOARD_GAME for the expansions. Workaround is to use excludeSubType=ThingType.BOARD_GAME_EXPANSION and make a 2nd call asking for subType=ThingType.BOARD_GAME_EXPANSION

Parameters

userName

Name of the user to request the collection for

subType

Specifies which collection you want to retrieve - only one type at the time

excludeSubType

Specifies which subtype you want to exclude from the results.

Filter collection to specifically listed item(s).

Returns version info for each item in your collection.

Returns more abbreviated results.
