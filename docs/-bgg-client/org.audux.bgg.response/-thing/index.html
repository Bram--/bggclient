data class Thing(val id: Int, val type: ThingType, val thumbnail: String?, val image: String?, val description: String?, val yearPublished: Int?, val datePublished: String?, val releaseDate: LocalDate?, val minPlayers: Int?, val maxPlayers: Double?, val playingTimeInMinutes: Int?, val minPlayingTimeInMinutes: Int?, val maxPlayingTimeInMinutes: Int?, val minAge: Int?, val videos: List<Video> = listOf(), val comments: Comments?, val statistics: Statistics?, val listings: List<MarketplaceListing> = listOf(), val versions: List<Version> = listOf(), val seriesCode: String?, val issueIndex: Int?, var name: String = "")

An item or thing which could be either of the org.audux.bgg.common.ThingType objects. As a result of the loose overlap of the types most values are Nullable, however type specific data is supplied via the Link property.

Furthermore, the settings/filling of the properties is highly dependent on the request. This is because additional parameters need to be set in order for the data to be retrieved. For example making a things request with simply an id parameter will not return any videos, comments, marketplace data, statistics or version information.
