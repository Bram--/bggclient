PaginatedRequest

abstract class PaginatedRequest<T> : Request<T>

Allows pagination on compatible API requests. Using paginate will automatically request all pages from page of the initial request. By default it paginates all pages but this can be controlled by the to parameter.

Parameters

client

The BggClient that's running this request

request

The initial request that should be paginated, it will paginate FROM the first page requested i.e. if the request makes a request with page = 10 pagination will only happen from 10 onwards.

abstract fun paginate(toPage: Int): Request<T>

Paginate all pages or only up to toPage, if set. If toPage is higher than the actual available pages no additional requests are made and instead will stop paginating after the final page was requests.
