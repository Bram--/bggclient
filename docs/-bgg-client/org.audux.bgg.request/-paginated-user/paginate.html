paginate

open override fun paginate(toPage: Int): <Error class: unknown class>(source)

Paginate all pages or only up to toPage, if set. If toPage is higher than the actual available pages no additional requests are made and instead will stop paginating after the final page was requests.

The returned T will have page set to the last page that was requested and T data class will have the paginated items copied in. i.e. when requesting 'plays' fpr a user with 300 plays the resulting Plays object will look something as follows:

Plays(total = 3, ...., page = 3, plays = List<Play>( /* 300 items in here */)
