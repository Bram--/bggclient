fun plays(username: String, id: Int? = null, type: PlayThingType? = null, minDate: LocalDate? = null, maxDate: LocalDate? = null, subType: SubType? = null, page: Int? = null): PaginatedPlays(source)

Request a list of plays (max 100 at the time) for the given user, returning org.audux.bgg.response.Plays.

Requesting a list of plays after 2022 for a specific username be done as follows:

Response<Plays> plays = BggClient
     .plays(username = "user", minDate = LocalDate.of(2023, 1, 1)).call()
println(plays.data) // Prints a Plays object with a list of (max 100) Play objects.

If the number of plays exceeds 100 not all would've been retrieved with the above API call. In order to do so pagination can be used:

Response<Plays> plays = BggClient
     .plays(username = "user", minDate = LocalDate.of(2023, 1, 1)).paginate().call()
println(plays.data) // Prints a Plays object with a list of ALL Play objects.

The actual class returned is org.audux.bgg.request.PaginatedPlays) which can then be used to paginate or make the actual call to the API using call or callAsync. This will then return a Response and wrap a data class (on success). e.g. the above pagination example might return a org.audux.bgg.response.Plays looking something as follows:

Plays(
     username = "user",
     userid="12345",
     total = 240,
     page = 1, // 1 if no pagination or last page when paginating
     plays = listOf(
         Play(
             id = 45678,
             date = LocalDate,
             quantity = 2,
             lengthInMinutes = 120,
             item = PlayItem(name = "Earth", objectId = 350184, /** ... */),
              players = listOf(),
          ),
          // etc...
     )
 )

Parameters

username

Name of the player you want to request play information for. Data is returned in backwards-chronological form. You must include either a username or an id and type to get results.

id

Id number of the item you want to request play information for. Data is returned in backwards-chronological form.

type

Type of the item you want to request play information for. Valid types include: PlayThingType.

minDate

Returns only plays of the specified date or later.

maxDate

Returns only plays of the specified date or earlier.

subType

=TYPE Limits play results to the specified TYPE; boardgame is the default.

page

The page of information to request. Page size is 100 records.
